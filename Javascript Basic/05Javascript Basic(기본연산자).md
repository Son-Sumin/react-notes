### 연산자(Operators)

#### 나머지(%) 사용처
- 홀짝 구분   
  홀수 : X % 2 = 1   
  짝수 : Y % 2 = 0   

- 특정 수보다 작은 수 구하기   
  특정 수를 5라고 하면   
  X % 5 = 0  => 0~4 값만 반환   

#### 거듭제곱
- const num = 2**3;   
  console.log(num);  // 8   

#### 연산자 우선순위
- * /  >  + -
- 연산자 줄여쓰기
<pre>
<code>
    let num = 10;
    num = num + 5;
    console.log(num);  // 15

    num += 5;  // 15
    num *= 5;  // 50
    num -= 5;  // 5
    num %= 5;  // 0
</code>
</pre>
<br>

#### 증가연산자(+1), 감소연산자(-1)
<pre>
